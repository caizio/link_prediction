### 基于python实现的链路预测框架
1. 复现了经典链路预测方法CN、JC、A3、Kataz
2. 实现了AUC和Precision指标
3. 本人提出了SMS和maxSMS链路预测方法（适用于互补性网络的链路预测，例如蛋白质相互作用网络）
